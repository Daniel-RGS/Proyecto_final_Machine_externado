Prediciendo la Escalada en Medio Oriente

Conflicto Irán–EE.UU.–Israel (2025-2026)

Proyecto Final ML1 - Universidad Externado de Colombia

El Reto 🎯

¿Podemos utilizar fuentes abiertas (OSINT) para anticipar aumentos de atención pública alrededor de un conflicto internacional?

Nuestra Hipótesis

La atención pública observable en fuentes abiertas puede funcionar como señal temprana de momentos de mayor tensión informativa.

Contexto 2025-2026 🗓️

  • 2025-2026: Seguimiento diario de atención pública en artículos asociados al conflicto regional.
  • Unidad de análisis: Día calendario.
  • Variable objetivo: Alta presión pública al día siguiente.

Fuentes de Datos 📡

🎯 Fuentes integradas

Wikimedia Pageviews + Wikipedia Revisions

Atención pública y actividad editorial diaria. Construimos un Score de Presión Pública (0-100).

RSS / Google News, BBC y Al Jazeera

Titulares recientes reales como fuente textual sugerida por el enunciado.

📊 Fuentes opcionales

GDELT DOC API

Volumen y tono noticioso si la API responde sin bloqueo o demoras excesivas.

ACLED / NASA FIRMS

Eventos y señales térmicas si se obtienen credenciales. No se usan reemplazos sintéticos.

Enfoque Machine Learning 🤖

Formulamos el reto como un problema de Clasificación Supervisada.

Baseline

Dummy Classifier

Regularización

Logistic Regression y Ridge Classifier

No-lineales

KNN Classifier & Random Forest

Métrica principal: F1 en holdout temporal

Arquitectura del Sistema 🏗️

  1. Ingesta: Scripts en Python automatizados (API REST & RSS Parsing).
  2. Procesamiento: Pandas & Numpy para limpieza, suavizado y creación de variables rezagadas (Lags temporales).
  3. Modelado: Scikit-Learn pipelines, escalamiento estandarizado y holdout temporal.
  4. Despliegue: Aplicación interactiva construida con Streamlit.

El Dashboard 🖥️

Una herramienta interactiva diseñada para analistas de inteligencia estratégica:

  • 📈 Timeline Histórico: Visualización interactiva del score diario de presión pública.
  • 🌐 Fuentes: Comparación de pageviews, revisiones, titulares RSS y fuentes opcionales disponibles.
  • 🔮 Predicción: Clasificación del riesgo de alta presión pública al día siguiente.

Conclusiones & Limitaciones 📌

Hallazgos

  • El modelo supera el baseline mayoritario en F1 dentro de una partición temporal.
  • Las señales de atención pública permiten detectar días de presión informativa alta.

Limitaciones

  • El modelo no predice violencia física; predice atención pública alta.
  • Ya se integran 3 fuentes reales; una cuarta fuente operacional como ACLED, FIRMS o UKMTO fortalecería la lectura de eventos en terreno.
